# EventSphere 🎟️✨

EventSphere is an AI-powered smart event management platform built for hackathons and modern event experiences.

It enables organizers to create and manage events while allowing attendees to discover, register, generate tickets, and validate entry using QR technology.

---

# 🚀 Features

## 👤 Authentication
- Firebase Authentication
- Organizer & Attendee roles
- Signup/Login system

## 🎉 Event Management
- Organizers can create events
- Dynamic event listing from Firestore
- Event details page

## 🎟️ Smart Ticketing System
- Ticket generation after registration
- Free & Paid event flows
- QR code ticket generation
- Ticket popup UI
- Download ticket as image

## 📱 QR Scanner Validation
- Organizer-side QR scanner
- Camera access support
- Real-time ticket validation
- Valid/Invalid attendee verification

## 🤖 AI Features
- AI Event Description Generator
- AI Event Recommendations

## 🎨 UI/UX
- Dark theme
- Purple + blue glassmorphism design
- Fully responsive layout
- Modern SaaS-inspired interface

# 🔥 Core Workflows

## Organizer Flow
1. Login/Signup
2. Create Event
3. Generate AI Description
4. Publish Event
5. Scan Attendee QR Tickets

## Attendee Flow
1. Browse Events
2. Open Event Details
3. Register/Buy Ticket
4. Ticket Generation
5. QR Ticket Popup
6. Download Ticket
7. Event Entry Validation
